:::key
(Semantics) The region of the code that a Binding takes effect.
:::
:::key
(Syntax & Grammar, Semantics) An **[Abstract binding tree (ABT)](/journeys/plt/concepts/abstract-binding-tree-abt)** *within which* the **bound identifier can be used**, either as a **placeholder** (in the case of a **variable declaration**) or as **the index of some Indexed operator** (in the case of a Symbol **declaration**).
:::
